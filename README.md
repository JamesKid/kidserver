#### Description
    It is a server status and control framework.

#### Target server
    1. 静态资源服务器
    2. 数据库服务器
    3. 缓存服务器

#### Target
    为负载均衡框架提供稳定可靠数据参考

    1. 获取服务器配置
    2. 获取服务器健康状况
    3. 获取服务器当前cpu,内存,带宽使用状况
    4. 统一调配服务器
    5. 全天服务器状态数据入库,曲线展示每小时服务器负载情况

#### status

#### control
    1. 安装新服务器常用软件
    2. 可选配置lnmp, gitlab服务器,zabbix 服务器
    3. 安装服务器状态监控框架
    4. 服务器安全配置
